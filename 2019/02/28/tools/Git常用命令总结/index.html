Git常用命令总结

初始化配置相关

  • 查看git的版本信息

    git --version

  • 查看当前登录的用户

    git config --global user.name

  • 查看当前登录的用户的邮箱

    git config --global user.email

    注意：如果在配置信息中没有获取到用户配置，说明没有登录。如果没有登录则只能拉取代码，不能修改，要是使用git，需要告诉git是谁在使用。

  • 设置登录用户

    git config --global user.name 'yourUserName'

  • 设置登录用户的邮箱

    git config --global user.email 'yourEmail'

  • 初始化git仓库

    git init

    注意：在gavin文件夹下创建了一个仓库，此时文件夹下会有一个.git的隐藏文件夹

文件操作相关

  • 创建文件夹

    mkdir gavin

  • 切换到文件夹

    cd gavin

  • 创建文件并写入内容

    echo "hello git" > index.html

    注意：单个箭头>表示覆盖写入，>>表示追加

  • 删除本地文件

    rm yourFileName

查询相关

  • 查看目录

    ls -al

  • 查看文件内容

    cat index.html

  • 查询仓库状态

    git status

  • 比较文件差异（请在git add之前使用）

    git diff 文件名

  • 查看仓库历史记录(详细)：

    git log

  • 查看仓库历史记录(单行)

    git log --pretty=online 或 git log --online

  • 查看所有版本的commit ID

    git reflog

添加相关

  • 添加指定文件到暂存区

    git add 文件名 （或git add -A 添加所有文件到暂存区）

    注意：add前需要先初始化仓库git init

  • 暂存区提交到仓库

    git commit -m "注释信息"

撤销相关

  • 撤销工作区的修改

    git checkout -- 文件名

  • 撤销暂存区的修改

    git reset HEAD 文件名

  • 回退到历史版本

    git reset --hard 该版本ID

  • 回退到上个版本

    git reset --hard HEAD^

    注意：上上版本是HEAD^^，也可用HEAD~2表示，以此类推

分支管理

  • 创建分支

    git branch dev

  • 切换分支

    git checkout dev

  • 创建分支并切换分支

    git checkout -b dev

  • 删除分支

    git branch -d dev

  • 在分支上提交新的版本

    git commit -a -m 'dev1'

  • 合并分支

    git merge dev

  • 分支合并后显示log

    git log  --oneline --graph --decorate

  • 在分支开发过程中遇到其他问题需要切换其他分支

    git　stash   //1.保留写好的内容
git checkout master //2.切换到主分支
git stash apply //3.应用保留下来的内容
git stash drop //4.可选：丢掉保留的内容
git stash pop //5.可选：使用保留的内容，并丢掉

  • 最佳分支

    git rebase

远程仓库

  • 先有本地库，后有远程库情况

    git remote add origin 网站上的仓库地址   //关联本地仓库和GitHub库
git push -u origin master    //第一次将本地仓库推送到GitHub上

  • 现有远程库，后有本地库情况

    git clone 网站上的仓库地址    //从远程库克隆到本地，可选https协议或ssh协议

  • 用本地库更新远程库

    git push origin master    //本地库的改动提交到远程库

  • 用远程库更新本地库

    git pull //更新本地库至远程库的最新改动

生成SSH Key

  • 生成SSH Key

    git-keygen -t rsa -C '你的邮箱@xx.com'

    注意：生成的SSH Key在”C:\用户\用户名.ssh”文件夹下
